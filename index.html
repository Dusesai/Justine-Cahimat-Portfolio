Projects

01 — System Design

E-Document Request & Resident Information System

The E-Document Request and Information System is a web-based solution designed to manage resident information and streamline document request processing. This project was developed in response to the inefficiencies of manual document handling in local offices, which are often unorganized and time-consuming. I contributed by designing system workflows, creating use case diagrams, and developing interface prototypes to visualize the system structure. Additionally, I implemented request tracking, validation processes, and a structured workflow to improve overall efficiency. The result is a more organized system concept that enhances tracking, reduces manual effort, and improves service delivery.

02 — Web Development

NeoFashion E-Commerce Website

NeoFashion is a responsive e-commerce platform inspired by modern online shopping interfaces, designed to deliver a smooth and engaging user experience across devices. The project was created to address the common issue of beginner e-commerce sites lacking responsive design and intuitive navigation. To solve this, I developed a fully responsive web interface focusing on clean UI design, structured product displays, and interactive components. As a result, the platform provides a seamless shopping experience with improved usability and consistent performance on both desktop and mobile devices.

03 — UI/UX & Visual Design

Human Flourishing Infographic

This project is a visual infographic that illustrates the evolution of human flourishing from traditional philosophical perspectives to modern global ideas. It was created to address the challenge of presenting complex concepts in a way that is easy to understand and engaging. Using Figma, I designed a structured and visually appealing layout that emphasizes clarity, hierarchy, and storytelling. The final output effectively simplifies abstract ideas into a clear and informative visual, making the content more accessible to a wider audience.
